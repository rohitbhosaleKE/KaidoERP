# Kaido ERP

Agentic AI powered ERP system with sales, inventory, CRM, and workflow automation.

## Features
- Sales management
- Inventory tracking
- Client management
- AI assistant
- Purchase & work order flow

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL/MySQL

## Run Locally

```bash
npm install
npm run dev
